Detailed Description

Socket compat, few utils.

Socket headers included:

  • win32: <winsock2.h>
  • win32: <ws2tcpip.h>
  • <sys/socket.h>
  • <sys/poll.h>
  • <sys/un.h>
  • <netinet/in.h>
  • <netinet/tcp.h>
  • <arpa/inet.h>
  • <fcntl.h>
  • <poll.h>
